Etiquetas Iniciales

DOCTYPE declaration

<!DOCTYPE html>

Esta es una declaración que indica al navegador web que el documento está escrito en HTML5, la última versión del lenguaje de marcado HTML. Es una declaración obligatoria y debe estar al comienzo del documento para asegurarse de que el navegador interprete el contenido de manera adecuada.

HTML root element

<html lang="es">

Esta etiqueta html es la envoltura principal de todo el documento HTML. El atributo lang especifica el idioma del contenido de la página, en este caso, se establece en español.

Head element

<head>

El elemento head es donde se coloca la información sobre el documento que no se muestra directamente en la página, como metadatos, enlaces a hojas de estilo (CSS), enlaces a scripts, títulos de la página y más.

Meta Charset

<meta charset="UTF-8">

Esta etiqueta meta especifica la codificación de caracteres utilizada en el documento. En este caso, se establece en UTF-8, que es una codificación que admite una amplia gama de caracteres de diferentes idiomas.

Viewport meta

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Esta etiqueta meta establece las propiedades de visualización en dispositivos móviles. El atributo content especifica que el ancho debe adaptarse al ancho del dispositivo (width=device-width) y que la escala inicial debe ser 1.0 (initial-scale=1.0).

Title element

<title>NotasHTML</title>

Esta etiqueta title define el título de la página, que se mostrará en la pestaña del navegador y en los resultados de búsqueda. En este caso, el título se establece como "NotasHTML".

Body element

<body>

El elemento body es donde se coloca el contenido visible de la página web. Aquí es donde se agregarían los encabezados, párrafos, imágenes, enlaces y otros elementos que los usuarios verán en el navegador.

Nota:

En resumen, estas etiquetas en conjunto conforman la estructura básica de un documento HTML. La declaración DOCTYPE asegura que se esté utilizando la versión correcta de HTML, el elemento html envuelve todo el contenido, el head contiene metadatos y enlaces a recursos, y el body contiene el contenido visible de la página.

Meta Etiquetas-SEO

Las metaetiquetas son elementos HTML que proporcionan información sobre el documento HTML. Son parte del <head> de una página web y no tienen contenido visible en la página, sino que contienen metadatos que describen la página y cómo debe ser tratada por los navegadores y las redes sociales.

Meta Description

< meta name="description" content="En esta pagina estamos aprendiendo los fundamentos de HTML con JonhMircha">

Esta metaetiqueta define la descripción del contenido de la página web. La descripción se utiliza en los resultados de búsqueda y en las vistas previas compartidas en redes sociales para brindar a los usuarios una breve idea de lo que pueden esperar encontrar en la página.

Canonical Link

<link rel="canonical" href="http://tudominio.com/la-url-canonica" >

Esta etiqueta especifica la URL canónica de la página. La URL canónica es la versión preferida de una página en situaciones en las que múltiples URLs pueden acceder al mismo contenido. Ayuda a los motores de búsqueda a entender cuál es la URL principal de una página.

Favicon

<link rel="icon" href="imagenes/FabiconHTML.png" >

Esta etiqueta establece el ícono del sitio (favicon) que se muestra en la pestaña del navegador y en los marcadores. El atributo href especifica la ubicación del archivo de ícono.

Apple Touch Icon

<link rel="apple-touch-icon" href="imagenes/FabiconHTML.png" >

Esta etiqueta define el ícono que se mostrará cuando los usuarios agreguen el sitio a la pantalla de inicio de su dispositivo movil, principalmente pensado para (iPhone o iPad).

Theme Color

<meta name="theme-color" content="#ff6600" >

Esta metaetiqueta define el color del tema de la barra de direcciones y otros elementos de interfaz del navegador cuando el sitio se guarda en la pantalla de inicio en dispositivos móviles.

Open Graph Title

<meta property="og:title" content="Aprendiendo Html" >

Establece el título que se mostrará cuando el contenido se comparta en plataformas de redes sociales como Facebook. El prefijo "og" significa "Open Graph", un protocolo utilizado por Facebook para compartir enlaces enriquecidos.

Open Graph Description

<meta property="og:description" content=" En esta pagina estamos aprendiendo los fundamentos de HTML con JonhMircha" >

: Define la descripción que se mostrará cuando se comparta el contenido en redes sociales.

Open Graph Image

<meta property="og:image" content="https://tudominio.com/imagenes/html5.jpg" >

Define la imagen que se mostrará cuando el contenido se comparta en redes sociales.

Open Graph URL

<meta property="og:url" content="https://tudominio.com/index.html" >

Especifica la URL que se compartirá en redes sociales.

Twitter Card

<meta name="twitter:card" content="summary" >

Define el tipo de tarjeta de Twitter que se mostrará cuando el contenido se comparta en Twitter.

Twitter Site

<meta name="twitter:site" content="@Nombre de la cuenta" >

Define la cuenta de Twitter asociada al sitio web.

<!--"Comentario que deseamos agregar"-- >

Los comentarios son secciones de texto que no se muestran en la página web cuando se carga en un navegador. En cambio, están destinados a proporcionar información adicional, explicaciones o notas para los desarrolladores o cualquier persona que revise el código.

Etiquetas de Encabezado

<h1>Encabezado 1</h1>

Las etiquetas de encabezado, desde <h1> hasta <h6>, se utilizan para definir los niveles de encabezados en una página web. <h1> es el nivel más alto y generalmente se usa para el título principal de la página. A medida que desciendes en los niveles (< h2>, < h3>, etc.), los encabezados se consideran subordinados a los niveles anteriores.Por lo general como normativa solo se utiliza un h1 por pagina web.

Etiqueta de Parrafo

<p>"texto del parrafo"</p>

La etiqueta <p> se utiliza para crear párrafos de texto en una página web. Todo el contenido dentro de la etiqueta <p> se considerará como un párrafo separado. Los navegadores automáticamente agregan espacio antes y después del párrafo para separarlo visualmente del contenido circundante.

Etiquetas de Texto basicas

Letra en Negrita

<b>negrita</b >

se utiliza para aplicar formato de texto en negrita. Cualquier texto contenido dentro de estas etiquetas se mostrará con un estilo de fuente en negrita. Sin embargo, el uso de esta solo cambia la apariencia visual del texto y no tiene un significado semántico específico. Para aplicar formato y significado, es preferible usar la etiqueta <strong > en su lugar.

Letra de tipo Italica

<i>italica</i >

La etiqueta <i> se usa para aplicar formato de texto en cursiva o itálica. El texto contenido dentro de estas etiquetas se mostrará en una fuente cursiva. Similar a <b>, si deseas aplicar significado semántico además del formato, es mejor usar la etiqueta <em>.

Letra subrayada

<u>"subrayado"</u >

se utiliza para aplicar un formato de subrayado al texto, lo que significa que una línea horizontal se dibuja debajo del contenido del texto. Es comúnmente utilizada para resaltar o enfatizar ciertas palabras o frases.

Etiquetas para dar elevacion a las letras

<sub>"elevacion de letras"</sub>

se usa para representar texto en formato de subíndice. En este ejemplo, se emplea para mostrar la famosa ecuación de la relatividad de Albert Einstein, donde el número "2" aparece ligeramente más abajo que el resto de los caracteres, indicando que es un exponente.Similar al ejemplo anterior, esta etiqueta HTML de subíndice se utiliza para mostrar la fórmula química del agua. El número "2" está desplazado hacia abajo para indicar que es un subíndice y se refiere a la cantidad de átomos de hidrógeno en la molécula.

Letras Pequeñas

<small>"Letras en formato pequeño"</small>

La etiqueta <small> se utiliza en HTML para reducir el tamaño del texto y presentar el contenido en una escala menor. Esto es útil cuando deseas mostrar información secundaria o menos relevante que no necesita ser resaltada en tamaño normal. En el ejemplo proporcionado, se usa para indicar que el texto "Letras pequeñas" debe ser mostrado en un tamaño de fuente más pequeño que el texto circundante.

Etiquetas de Texto-Semanticas

Texto Importante

<strong >" Texto importante "</strong >

La etiqueta <strong > se utiliza para resaltar texto y darle importancia semántica, generalmente aplicando un formato en negrita. No solo cambia el aspecto visual del texto, sino que también indica que el contenido es significativo o relevante en términos de estructura del documento.

Texto de Enfasis

<em>"Este texto hace enfasis"</em>

La etiqueta <em> se emplea para aplicar énfasis a un texto, generalmente mediante el uso de cursiva. Indica que el contenido es enfatizado, a menudo para transmitir un tono o significado especial.

Bloque de Texto

<blockquote >" "</ blockquote>

La etiqueta <blockquote > se utiliza para formatear un bloque de texto que es una cita o una sección de texto que se toma de otra fuente. El contenido dentro de esta etiqueta se suele mostrar con un margen adicional en ambos lados. En tu ejemplo, se utiliza para representar una cita textual.

<cite >" Fuente citada "</cite >

La etiqueta <cite > se utiliza para citar la fuente de una obra creativa, como un libro, una película, un poema, etc. Se usa generalmente en combinación con elementos que describen la obra citada. En tu caso, "Abuelo Daniel" es la fuente citada.

Etiquetas de Saltos

Salto de Linea

<br>

La etiqueta <br> se utiliza para realizar un salto de línea dentro de un bloque de texto. No requiere una etiqueta de cierre porque es una etiqueta de autocierre. Es útil cuando deseas forzar un salto de línea en un lugar específico, como en una dirección o en un poema.

Salto de Linea, Seccion

<hr>

La etiqueta <hr> se utiliza para insertar una línea horizontal en una página web. Crea una separación visual entre secciones de contenido. Al igual que la etiqueta <br>, es una etiqueta de autocierre y no necesita una etiqueta de cierre.

🛦

Etiquetas de Formateo

Texto preformateado en bloque

<pre >"Bloque de texto"</ pre>

La etiqueta <pre > se utiliza para mostrar texto preformateado en un bloque. Esto significa que el texto se muestra tal como está en el código fuente, manteniendo los espacios en blanco y saltos de línea. Es especialmente útil para mostrar código o bloques de texto que requieren un formato específico.

Texto tipo Codigo

<code >"Letra tipo codigo "</code >

La etiqueta <code > se utiliza para resaltar una porción de código o texto en la fuente monoespaciada (como la que se usa en el código) para indicar que es parte del código o la sintaxis.

Etiquetas Semanticas Estructurales

Etiqueta div

<div>" Contenido "</div>

La etiqueta <div> se utiliza como un contenedor genérico para agrupar y estructurar contenido en bloques. No tiene un propósito semántico específico, pero es ampliamente utilizada para aplicar estilos y dividir la página en secciones lógicas.

Etiqueta de Encabezado

<header>" Encabezado "</header>

La etiqueta <header> se utiliza para representar la sección de encabezado de una página o sección. Por lo general, contiene elementos como el logotipo, el título de la página, enlaces de navegación principales u otros elementos que son parte esencial de la cabecera de la página.

Contenido Principal

<main>" contenido de principal de la pagina "</main>

La etiqueta <main> se utiliza para encapsular el contenido principal del documento. En una página web, debe haber un solo elemento

que contenga el contenido principal que los usuarios están buscando, como artículos, secciones principales u otros elementos clave.

Pie de Pagina

<footer >"Final del Texto"</footer >

La etiqueta <footer > se utiliza para representar la sección de pie de página de una página o sección. Por lo general, contiene información como derechos de autor, enlaces a páginas relacionadas, información de contacto o cualquier contenido que sea relevante para el pie de página.

Navegacion

<nav >"Representa una navegacion "</nav >

La etiqueta <nav > se utiliza para encapsular los enlaces de navegación, como menús o listas de enlaces que ayudan a los usuarios a moverse por el sitio web. Ayuda a identificar áreas de contenido que se consideran como elementos de navegación.

Articulos

<article >"Representa una seccion de autocontenido "</article >

La etiqueta <article > se utiliza para representar un contenido independiente y autónomo en una página. Puede ser un artículo de noticias, una publicación de blog, una entrada de foro, etc. Debe tener sentido por sí mismo y ser capaz de distribuirse y reutilizarse independientemente del resto del contenido.

Contenido Complementario o Secundario

<aside>"Representa contenido complementario o secundario "</aside>

La etiqueta <aside> se utiliza para encapsular contenido que es complementario o secundario al contenido principal de la página. Puede contener información relacionada, anuncios, widgets, enlaces relacionados o cualquier contenido no esencial para el flujo principal de la página.

Secciones

<section >"Representa una seccion de contenido generico "</section >

La etiqueta <section > se utiliza para agrupar contenido relacionado en secciones temáticas. Ayuda a dividir y organizar el contenido de manera semántica, lo que facilita la comprensión y la estructura del documento.

Direcciones

<address >"Representa una direccion "</address >

La etiqueta <address > se utiliza para encapsular información de contacto, como una dirección de correo electrónico, una dirección postal o información de contacto similar. Puede ser utilizado en el pie de página o dentro del contenido de un artículo o sección para indicar información de contacto relevante.

Etiquetas de Bloque VS Etiquetas de linea

<div > es la Etiqueta de Bloque por Excelencia

<div >"Etiqueta de Bloque "</div >

La etiqueta <div > en HTML se utiliza como un contenedor genérico y versátil para agrupar y dividir el contenido en bloques o secciones. No tiene un propósito semántico en sí misma, pero se utiliza para aplicar estilos y estructurar la página web. Puedes aplicar estilos CSS, como márgenes, rellenos y bordes, a elementos <div > para crear diseños complejos y organizar el contenido. Es útil cuando deseas crear secciones o áreas específicas en tu página web y aplicar estilos o manipulaciones al contenido dentro de esas áreas.

<span > es la Etiqueta de Linea por Excelencia

<span >"Etiqueta de Linea "</sapn>

La etiqueta <span > se utiliza para aplicar estilo o marcar un fragmento específico de texto dentro de un elemento en línea, como un párrafo. A diferencia de <div >, que es un contenedor de bloque, <span > es un contenedor en línea. Esto significa que no cambia la estructura de línea y solo afecta al fragmento de texto contenido en su interior. Puedes aplicar estilos como colores de texto, resaltado o incluso clases CSS a elementos <span >. Es útil cuando deseas aplicar estilos o efectos a partes específicas de un texto en línea sin afectar a todo el párrafo.

Estilos CSS en HTML

Aplicando Estilos con la Etiqueta div

<div style="background-color:green;color:aqua">"Esta es una de las formas para aplicar estilos "</div >

En este ejemplo, estás aplicando estilos directamente en línea utilizando el atributo style en la etiqueta <div >. El atributo style te permite definir reglas de estilo en línea para un elemento HTML específico. En este caso, el fondo del <div > tendrá un color verde (background-color:green) y el texto dentro del <div > será de color aguamarina (color:aqua).

Aplicando Estilos desde el encabezado

<style >"h2{color:brown;}"</style >

De esta manera aplicamos los estilos desde el encabezado de nuestro documento HTML

Aplicando Estilos con la Etiqueta span

<span style="background-color:cadetblue" >"De esta forma aplicamos estilos n¿con la etiqueta span "</span >

Aquí, también estás aplicando estilos en línea usando el atributo style, pero esta vez para la etiqueta <span >. El fondo del <span > será de color azul cadete (background-color:cadetblue).

Aplicando los Estilos desde una hoja

<link rel="stylesheet" href="assets/estilos.css" >

Esta es una forma diferente y más organizada de aplicar estilos a través de un archivo de hoja de estilos externo. Estás enlazando una hoja de estilos ubicada en el archivo "estilos.css" en la carpeta "assets". Esta hoja de estilos contiene reglas de estilo que afectarán a múltiples elementos en tus páginas web. Esta forma es preferible cuando deseas mantener tus estilos separados del contenido HTML, lo que facilita la gestión y el mantenimiento de los estilos en todo el sitio web.

Etiquetas de Imagenes

Imagenes

<img src="imagenes/spiderman1.jpg" alt="Spiderman3" >

La etiqueta <img> se utiliza para insertar imágenes en una página web. El atributo src especifica la fuente de la imagen, y el atributo alt proporciona un texto alternativo que se muestra si la imagen no puede cargarse o en caso de que sea leída por dispositivos de asistencia para accesibilidad. En este caso, la imagen "spiderman1.jpg" se mostrará y si no se carga, se mostrará el texto alternativo "Spiderman3".

Imagenes SVG

<img src="imagenes/hero-solid.svg" alt="imagen en formato SVG django" >

imilar al ejemplo anterior, esta etiqueta <img > está insertando una imagen en formato SVG. Los archivos SVG son gráficos vectoriales escalables que se pueden mostrar a cualquier tamaño sin perder calidad. El atributo alt proporciona una descripción alternativa en caso de que la imagen no pueda cargarse.

Figuras

<figure > 
  <img src="imagenes/halo.jpg" alt="Imagenes de Halo"> 
       <figcaption >"descripcion de nuestra imagen "</figcaption > 
       </figure >

La etiqueta <figure > se utiliza para agrupar contenido multimedia, como imágenes, y proporcionar un contexto semántico para ese contenido. Dentro de <figure >, se coloca una <img> para mostrar la imagen y un <figcaption > para proporcionar una descripción o título para la imagen. Esto ayuda a relacionar el contenido visual con su contexto descriptivo.

Listas Ordenadas

Listas de Orden normal


  <ol> 
   <li> Primavera </li> 
   <li> Verano </li> 
   <li> Otoño </li> 
   <li> Invierno </li> 
  </ol>

La etiqueta <ol > se utiliza para crear una lista ordenada. Los elementos <li > (elementos de lista) dentro de <ol > representan los elementos de la lista que estarán numerados automáticamente en orden. En este caso, se muestra una lista de las estaciones del año.

Lista con el atributo start


  <ol start="3"> 
   <li> Motos </li> 
   <li> Autos </li> 
   <li> Aviones </li> 
   <li> Lanchas </li> 
  </ol>

Aquí, el atributo start en la etiqueta <ol > se utiliza para establecer el número de inicio de la lista. En lugar de comenzar desde 1, la lista comenzará desde el número 3. Los elementos de la lista se numerarán en orden a partir de ese número.

Listas con el atributo reversed


  <ol reserved> 
   <li> Espadas </li> 
   <li> Dagas </li> 
   <li> Hachas </li> 
   <li> Lanzas </li> 
  </ol >

La utilización del atributo reversed en la etiqueta <ol > invierte el orden de numeración de la lista. Los elementos de la lista se numerarán en orden inverso, desde el último elemento al primero.

Listas con otra enumeracion


  <ol type="A"> 
   <li> Oro </li> 
   <li> Plata </li> 
   <li> Diamante </li> 
   <li> Perlas </li> 
  </ol>

El atributo type en la etiqueta <ol > permite cambiar la forma en que se enumeran los elementos de la lista. En este caso, type="A" establece que los elementos de la lista se enumerarán utilizando letras mayúsculas (A, B, C, etc.).

Listas de Definicion


  <dl> 
   <dt> HTML </dt> 
   <dd> Es un lenguaje de marcado que define el contenido de la web </dd>

La etiqueta <dt> se utiliza para definir un término en una lista de definición. En este caso, "html" se define como el término y la etiqueta <dd> se utiliza para proporcionar la definición de un término en una lista de definición. Aquí, se proporciona la definición de "html".

Esta es una tabla más estructurada. Utiliza las etiquetas <thead >, <tbody >, y <tfoot > para separar el encabezado, el cuerpo y el pie de la tabla, respectivamente. También utiliza la etiqueta <th> para definir encabezados de columna en lugar de datos de celda. El atributo colspan se utiliza para controlar cuántas columnas ocupa una celda en una fila. En resumen, las etiquetas <table>, <tr >, <td >, <thead >, <tbody >, <tfoot >, y <th> se utilizan para crear y estructurar tablas en HTML. Puedes utilizar estas etiquetas para presentar datos tabulares de manera organizada y fácil de leer en tu página web.

Data Atributos

Atributos perzonalizados

Los atributos de datos (o "data atributos") son atributos personalizados que se pueden añadir a elementos HTML para almacenar información adicional que no tiene un propósito visible directo en la presentación, pero que puede ser útil para scripts, estilos u otros procesos de manipulación de datos en la página web.Los atributos de datos se definen utilizando el prefijo "data-" seguido de un nombre que describe la información que se va a almacenar. Por ejemplo:

<ul> 
  <li data-id="1" data-name="spring">Primavera</li> 
  <li data-id="2" data-name="summer">Verano</li> 
  <li data-id="3" data-name="autumn">Otoño</li> 
  <li data-id="4" data-name="winter">Invierno</li> 
</ul>

data-id: Este atributo almacena el ID numérico asociado con la estación. En este caso, el valor es "1" para la primavera. data-name: Este atributo almacena el nombre de la estación. En este caso, el valor es "spring" para la primavera. Primavera: Es el contenido del elemento de lista, que en este caso es el nombre de la estación. Cada uno de los elementos de lista en la estructura tiene sus propios atributos data-id y data-name, que permiten almacenar información personalizada asociada a cada estación. Estos atributos personalizados son útiles cuando se necesita almacenar datos adicionales en los elementos HTML para su posterior uso en scripts, estilos u otros procesos.
