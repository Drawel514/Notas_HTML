Aprendiendo CSS

Cascading Style Sheets o Hojas de estilos en cascada es un lenguage de definicion de estilos que da presentacion para documentos HTML.

Regla CSS

Consta de dos partes:
1.El selector, es el elemento html al que podemos aplicarle ese bloque de declaracion de estilos.
2.Bloque de declaraciones que son cada uno de los atributos que vamos a modificar, su sintaxis es la siguiente (atrubuto: valor;)los cuales estan definidos dentro de llaves{} Por Ejemplo: 


    h1{
      color:oranged,
      background-color:skyblue;

Formas de Aplicar estilos

Aplicando estilos en linea

"Cascading Style Sheets"

Lo implementamos de la sigueinte manera, ya que se aplica en la linea de codigo a la cual queremos dar el estilo en espesifico.

Aplicando estilos Internamente

Lo cual significa que vamos a dar estilos desde el interior del codumento HTML, esto se implementa al inicio del documento y se hace de la siguiente manera: 

    <style> 
      h1{
        color:yellowgreen
      }
    <style/>


Aplicando estilos desde una hoja de estilos

  <link rel="stylesheet" href="assets/style.css" >

Esta es una forma diferente y más organizada de aplicar estilos a través de un archivo de hoja de estilos externo. Estás enlazando una hoja de estilos ubicada en el archivo "style.css" en la carpeta "assets". Esta hoja de estilos contiene reglas de estilo que afectarán a múltiples elementos en tus páginas web. Esta forma es preferible cuando deseas mantener tus estilos separados del contenido HTML, lo que facilita la gestión y el mantenimiento de los estilos en todo el sitio web.


Importando otra hoja de estilos en un ahoja de estilos(Aunque suene redundante asi es...)

Para realizar este procedimiento hacemos lo siguiente: 

@import url("OtrosStyles.css");
Nota:Debemos tener en cuenta que utilizar el @import no hace parte de una buena practica.Esto se debe a que son bloqueantes en el momento en el que el navegador lee la instruccion.

Selectores Basicos

Existen 3 selectores basicos los cules son:

1.Etiquetas
2.Identificadores
3.Clases


Selector de Etiquetas

El selector de etiquetas es el más simple y directo. Se utiliza para aplicar estilos a un tipo específico de elemento HTML. Por ejemplo, si deseas aplicar un estilo a todos los párrafos de tu página, puedes usar el selector de etiquetas para seleccionar el elemento <p>. Aquí hay un ejemplo:

    p {
      color: blue;
      font-size: 16px;
    }

Selector de Identificadores

Los identificadores (IDs) son atributos únicos que se pueden asignar a un elemento HTML. Los IDs se utilizan para identificar un elemento específico en el documento. Los selectores de identificadores se denotan con un signo de almohadilla (#) seguido del nombre del ID. Por ejemplo, si tienes un elemento con el ID "mi-elemento", puedes aplicar estilos a él de la siguiente manera:

    #mi-elemento {
      background-color: yellow;
      border: 1px solid black;
    }

Selector de Clases

Las clases son atributos que se pueden aplicar a múltiples elementos HTML. A diferencia de los IDs, las clases pueden ser compartidas por varios elementos. Los selectores de clases se denotan con un punto (.) seguido del nombre de la clase. Por ejemplo, si tienes varios elementos con la clase "destacado", puedes aplicar estilos a todos ellos de esta manera:

    .destacado {

Selectores Avanzados

Selectores de Hijos Directos y Descendientes

Selector de hijo directos

Un selector de hijo directo (>) en CSS se utiliza para seleccionar elementos que son hijos directos de un elemento específico. Esto significa que solo los elementos que son directamente anidados dentro del elemento padre serán seleccionados. Los elementos más profundos en la jerarquía no serán afectados.

  • YHWH
  • Elohim
  • Adonai
  • El Shadai

En este conjunto de reglas, se está utilizando el selector de hijo directo (>) para seleccionar todos los elementos <li> que son hijos directos de elementos con la clase .hijos-directos. Luego, se les aplica un fondo de color rojo oscuro (crimson).

  Estilo:
    .hijos-directos > li {
      background-color: crimson;
    }

  Ejemplo anterior:
    <ul class="hijos-directos">
      <li><b>YHWH</b></li>
      <li><b>Elohim</b></li>
      <li><b>Adonai</b></li>
      <li><b>El Shadai</b></li>
    </ul>

Selector de hijos descendientes

Un selector de hijo descendiente (espacio en blanco) en CSS selecciona todos los elementos que son descendientes de otro elemento, sin importar si son hijos directos o anidados a mayores niveles. Los elementos pueden estar anidados a cualquier profundidad dentro del elemento padre.

  • YHWH
  • Elohim
  • Adonai
  • El Shadai

En este conjunto de reglas, se está utilizando el selector de hijos descendientes (espacio en blanco) para seleccionar todos los elementos que son descendientes (ya sea hijos directos o anidados) de elementos con la clase .hijos-descendientes. Luego, se les aplica un fondo de color turquesa oscuro (darkturquoise).

    Estilo:
    .hijos-descendientes b{
      background-color: darkturquoise;
    }

    Ejemplo anterior:
    <ul class="hijos-descendientes">
      <li><b>YHWH</b></li>
      <li><b>Elohim</b></li>
      <li><b>Adonai</b></li>
      <li><b>El Shadai</b></li>
    </ul>

Selector de Hermanos

Selector de hermanos General y Adyacente

Seletor de hermanos General

En CSS, el selector de hermanos general (~) se utiliza para seleccionar elementos que comparten el mismo nivel dentro de su padre y que siguen al elemento de referencia. En tu código, el elemento de referencia es un elemento con la clase hermanos-generales. El estilo se aplicará a todos los elementos <b> que son hermanos del elemento con la clase hermanos-generales (incluido él mismo).

  • Pintura
  • Escultura
  • Dibujo
  • Puntillismo
Ejemplo: 
    Estilo:
    .hermanos-generales ~ li{
      background-color: gray;
    }

    Ejemplo anterior:
    <ul>
      <li><b>Pintura</b></li>
      <li class="hijos-descendientes"><b>Escultura</b></li>
      <li><b>Dibujo</b></li>
      <li><b>Puntillismo</b></li>
    </ul>

Selector de hermanos Adyacente

el selector de hermanos adyacente (+) se utiliza para seleccionar el primer elemento que es inmediatamente precedido por otro elemento. En tu código, el elemento precedente es un elemento con la clase hermanos-adyacentes. El estilo se aplicará solo al primer elemento <b> que sigue inmediatamente al elemento con la clase hermanos-adyacentes.

  • Pintura
  • Escultura
  • Dibujo
  • Puntillismo
Ejemplo: 
    Estilo:
    .hermanos-adyacentes + li{
      background-color: goldenrod;
    }

Selectores de Atributo

Los selectores de atributos son una parte importante de las hojas de estilo en cascada (CSS) que permiten seleccionar elementos HTML en función de los valores de sus atributos. Estos selectores te permiten aplicar estilos a elementos que cumplen ciertos criterios relacionados con sus atributos, como clases, identificadores, valores de atributos, etc. Los selectores de atributos son útiles cuando deseas aplicar estilos a elementos específicos sin depender únicamente de su posición en la estructura del documento HTML.

  • inicio
  • Cursos
  • blog
  • cv
  • ahora
  • Notas
  • Ir a Notas HTML

Ejemplo Anterior: 
    <h2>Selectores de Atributo</h2>
    <nav class="selectores-atributos">
        <ul>
          <li><a href="https://jonmircha.com" target="_blank">inicio</a></li>
          <li><a class="menu-item cursos button" href="https://jonmircha.com/cursos" target="_blank">Cursos</a></li>
          <li><a href="http://jonmircha.com/blog" target="_blank">blog</a></li>
          <li><a href="http://jonmircha.com/cv" target="_blank">cv</a></li>
          <li><a href="https://jonmircha.com/ahora" target="_blank">ahora</a></li>
          <li><a href="https://jonmircha.com/notas" target="_blank">Notas</a></li>
          <li><a href="Notas.html#inicio">Ir a Notas HTML</a></li>
        </ul>
    </nav>

Estilo 1:

.selectores-atributos a[href]: Este estilo se aplica a todos los elementos de anclaje (<a>) que tengan un atributo href. En este caso, el estilo cambia el color del texto a "cornflowerblue" (un tono de azul).

    .selectores-atributos a[href]{
      color: cornflowerblue;
    }

Estilo 2:

.selectores-atributos a[target="_blank"]: Este estilo se aplica a los elementos de anclaje () que tengan un atributo target con el valor "_blank" (lo que significa que se abrirán en una nueva pestaña o ventana del navegador). El estilo cambia la familia de fuentes del texto a 'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif.

    .selectores-atributos a[target="_blank"]{
      font-family: 'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif;
    }

Estilo 3:

.selectores-atributos a[href*="jonmircha"]: Este estilo se aplica a los elementos de anclaje () cuyo atributo href contiene la subcadena "jonmircha". El estilo cambia el tamaño de fuente del texto a 24 píxeles.

    .selectores-atributos a[href*="jonmircha"]{
      font-size: 24px;
    }

Estilo 4:

.selectores-atributos a[href^="http:"]: Este estilo se aplica a los elementos de anclaje () cuyo atributo href comienza con la cadena "http:". El estilo elimina la decoración de texto subrayado (la línea debajo del enlace).

    .selectores-atributos a[href^="http:"]{
      text-decoration: none;
    }

Estilo 5:

.selectores-atributos a[href$=".com"]: Este estilo se aplica a los elementos de anclaje () cuyo atributo href termina con la cadena ".com". El estilo cambia el color de fondo del texto a negro.

    .selectores-atributos a[href$=".com"]{
      background-color: black;
    }

Estilo 6:

.selectores-atributos a[class~="cursos"]: Este estilo se aplica a los elementos de anclaje () que tienen la clase "cursos" como una de las clases en su atributo class. El estilo cambia el color de fondo del texto a "darkslategrey" (un tono oscuro de gris).

    .selectores-atributos a[class~="cursos"]{
      background-color: darkslategrey;
    }

Estilo 7:

.selectores-atributos a[class|="menu"]: Este estilo se aplica a los elementos de anclaje () que tienen la clase que comienza con "menu" en su atributo class, seguido de un guión o siendo la única clase. El estilo cambia el estilo de fuente del texto a cursiva (italic).

    .selectores-atributos a[class|="menu"]{
      font-style: italic;
    }

Selector Universal(*)

Un selector universal en CSS es un selector especial que coincide con todos los elementos en un documento HTML. Se representa mediante el asterisco (*). Cuando se utiliza un selector universal, se aplicará a cada elemento dentro del ámbito en el que se encuentra. No tiene en cuenta el tipo de elemento, la etiqueta, la clase, el identificador u otros atributos específicos del elemento; simplemente selecciona todos los elementos.

  • Genesis
  • Exodo
  • Levitico
  • Numeros
  • Deuteronomio

    <ul class="selector-universal">
      <li><b>Genesis</b></li>
      <li><b>Exodo</b></li>
      <li><b>Levitico</b></li>
      <li><b>Numeros</b></li>
      <li><b>Deuteronomio</b></li>
    </ul>

Ejemplo 1:

* { font-family: 'Times New Roman', Times, serif; }: Este estilo se aplica a todos los elementos del documento HTML. La propiedad font-family establece la fuente del texto para estos elementos. Estás definiendo que la fuente sea 'Times New Roman', y si no está disponible, se use Times, y si ambas fuentes no están disponibles, se utilizará una fuente genérica de la familia serif. Esto significa que todos los elementos, incluidos los encabezados, párrafos, listas, etc., tendrán esta fuente.

    *{
      font-family: 'Times New Roman', Times, serif;
    }

Ejemplo 2:

.selector-universal * { font-size: 30px; }: Este estilo se aplica a todos los elementos dentro de cualquier elemento con la clase "selector-universal". Aquí, estás ajustando el tamaño de fuente de estos elementos internos a 30px. En este caso, los elementos internos son los elementos <li> dentro de la lista desordenada <ul>.

      .selector-universal *{
        font-size: 30px;
      }

Pseudoclases

Una pseudoclase CSS es una palabra clave que se añade a los selectores y que especifica un estado especial del elemento seleccionado.



    Codigo del Formulario anterior: 

    <form class="form-pseudoclases">

      <input type="text" name="nombre" placeholder="Escribe tu nombre" required>
      
      <input type="email" name="correo" placeholder="Escribe tu correo" required>
      
      <input type="submit">
    </form>

Ejemplo 1:

.form-pseudoclases *:required:Este estilo se aplica a cualquier elemento dentro del formulario con la clase form-pseudoclases que tenga el atributo required. Cuando un campo obligatorio (con el atributo required) no se ha completado, el borde de ese campo se coloreará en amarillo (#ffff01), indicando que debe ser llenado.

    .form-pseudoclases *:required{
      border-color: #ffff01;
    }

Ejemplo 2:

.form-pseudoclases *:valid:Este estilo se aplica a cualquier elemento dentro del formulario con la clase form-pseudoclases que esté en un estado válido. Cuando el valor de un campo es válido (por ejemplo, un campo de correo electrónico con un formato correcto), el borde de ese campo se coloreará en verde chartreuse.

    .form-pseudoclases *:valid{
      border-color: chartreuse;
    }

Ejemplo 3:

.form-pseudoclases *:invalid:Este estilo se aplica a cualquier elemento dentro del formulario con la clase form-pseudoclases que esté en un estado inválido. Cuando el valor de un campo es inválido (por ejemplo, un campo de correo electrónico con un formato incorrecto), el borde de ese campo se coloreará en rojo oscuro (#d93300).

    .form-pseudoclases *:invalid{
      border-color: #d93300;
    }

Ejemplo 4:

.form-pseudoclases *:focus:Este estilo se aplica a cualquier elemento dentro del formulario con la clase form-pseudoclases que esté en estado de enfoque (cuando se hace clic en él o se selecciona usando el teclado). Cuando un campo está en foco, el fondo de ese campo se coloreará en gris oscuro (darkgray).

    .form-pseudoclases *:focus{
      background-color: darkgray;
    }

Pseudoclase aplicada mediante ID

Este estilo se aplica al elemento con el ID inicio cuando se convierte en el fragmento de destino (target) de la URL. La pseudo-clase :target se utiliza para seleccionar un elemento que se encuentra enlazado desde otra parte de la página mediante un enlace de anclaje, es decir, cuando el usuario hace clic en un enlace que apunta a ese elemento específico. El estilo establecido en este caso cambia el color de fondo del elemento con el ID inicio a un tono de cian oscuro (darkcyan) cuando ese elemento se convierte en el fragmento de destino de la URL.

      #inicio:target{
      background-color: darkcyan;
    }

Agrupacion de Selectores

La agrupación de selectores en CSS es una técnica que te permite aplicar el mismo conjunto de estilos a múltiples elementos en una sola declaración. En lugar de repetir el mismo conjunto de propiedades y valores para cada selector individualmente, puedes agruparlos en una única regla de estilo para simplificar y optimizar tu código CSS.




Codigo anterior:

    <form class="form-agruparSelectores">
      <input type="text" name="nombre" placeholder="Escribe tu nombre">
      <br>
      <input type="email" name="correo" placeholder="Escribe tu correo">
      <br>
      <textarea name="comentarios" placeholder="Escribe un comentario" cols="30" rows="10"></textarea>
      <br>
      <input type="submit">
    </form>

Los estilos .form-agruparSelectores input[type="text"], .form-agruparSelectores input[type="email"], .form-agruparSelectores textarea se aplican a los campos de entrada (<input>) de tipo "text", los campos de entrada de tipo "email" y las áreas de texto (<textarea>) dentro del formulario con la clase form-agruparSelectores.

Estilos: 

    .form-agruparSelectores input[type="text"],
    .form-agruparSelectores input[type="email"],
    .form-agruparSelectores textarea{
      border-color: #0015ff;
      border-width: 3px;
      border-style: dotted;
      background-color: black;
      font-size: 20px;
      display:block
    }
  • (border-color: #0015ff;), (border-width: 3px;), (border-style: dotted;):Estos estilos combinados aplican un borde de color azul (#0015ff) con un ancho de 3 píxeles y un estilo de línea punteada a los campos de entrada y áreas de texto.
  • (background-color: black;):Este estilo establece un fondo negro para los campos de entrada y áreas de texto.
  • (font-size: 20px;):Este estilo establece un tamaño de fuente de 20 píxeles para el texto en los campos de entrada y áreas de texto.
  • (display: block;):Este estilo establece el valor "block" para el atributo display, lo que hace que cada campo de entrada y área de texto se muestre en una línea separada, es decir, cada uno ocupará todo el ancho disponible en su contenedor.

Resteo y Normalizacion de Estilos

El reseteo y la normalización son dos enfoques comunes en CSS para establecer una base uniforme y consistente en el diseño y estilo de los elementos HTML en diferentes navegadores. Ambos enfoques buscan solucionar problemas de inconsistencias y diferencias de diseño entre navegadores al proporcionar reglas de estilo iniciales.


Reseteo (CSS Reset)

El reseteo es una técnica que consiste en eliminar o restablecer los estilos predeterminados que los navegadores aplican a los elementos HTML. Los navegadores tienen hojas de estilo predeterminadas que pueden variar, lo que puede conducir a diferencias en la apariencia de los elementos en diferentes navegadores. Un reseteo busca nivelar el campo de juego al eliminar estos estilos predeterminados.Un ejemplo de reseteo podría ser:

    /* CSS Reset */
    * {
        margin: 0;
        padding: 0;
        box-sizing: border-box;
      }

En este ejemplo, se establece que todos los elementos (*) no tendrán márgenes ni rellenos por defecto y se utiliza el modelo de caja border-box, que garantiza que el ancho y alto incluyan los márgenes y rellenos.



Normalización (CSS Normalize)

La normalización es una técnica similar al reseteo, pero en lugar de eliminar todos los estilos predeterminados, busca mantener algunos estilos útiles mientras que normaliza otros para que funcionen de manera coherente en todos los navegadores. El objetivo es lograr una apariencia uniforme pero conservar algunas de las ventajas de los estilos predeterminados.

Un ejemplo de normalización sería utilizar una biblioteca como "Normalize.css", que está diseñada para corregir y mejorar estilos específicos en diferentes navegadores. Puedes incluir Normalize.css en tu proyecto y se encargará de establecer una base más consistente.

Prefijos de los Navegadores

os prefijos de los navegadores son un conjunto de letras o palabras que se agregan como prefijo a las propiedades CSS experimentales antes de que se estandaricen oficialmente. Estos prefijos permiten a los desarrolladores probar y utilizar características nuevas de CSS antes de que estén completamente soportadas en todos los navegadores, pero también indican que esas características aún están en proceso de desarrollo y podrían cambiar en futuras versiones.

Los prefijos de los navegadores ayudan a evitar problemas de compatibilidad y permiten a los desarrolladores utilizar características avanzadas antes de que se conviertan en parte de las especificaciones oficiales de CSS. Sin embargo, debido a que estas características pueden variar entre navegadores y versiones, se recomienda su uso con precaución y generalmente solo para casos en los que sea necesario.

Los prefijos de los navegadores generalmente se agregan en forma de un prefijo específico al comienzo de una propiedad CSS experimental. Aquí están algunos de los prefijos más comunes y sus correspondencias con navegadores:

  • Navegadores de Microsoft (-ms-)
  • FireFox (-moz-)
  • Todos los demas navegadores(-webkit-)
  • Opera (-o-)
