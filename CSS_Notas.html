Aprendiendo CSS

Cascading Style Sheets o Hojas de estilos en cascada es un lenguage de definicion de estilos que da presentacion para documentos HTML.

Regla CSS

Consta de dos partes:

1.El selector, es el elemento html al que podemos aplicarle ese bloque de declaracion de estilos.

2.Bloque de declaraciones que son cada uno de los atributos que vamos a modificar, su sintaxis es la siguiente (atrubuto: valor;)los cuales estan definidos dentro de llaves{}
Por Ejemplo:

  h1{
    color:oranged,
    background-color:skyblue;
  }

Aplicando estilos Internamente 

  Lo cual significa que vamos a dar estilos desde el interior del codumento HTML, esto se implementa al inicio del documento y se hace de la siguiente manera:


  <style>
